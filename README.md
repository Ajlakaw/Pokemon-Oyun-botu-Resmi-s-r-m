# Gelişmiş Discord Pokemon Botu

## Tanım:
### Bu gelişmiş bir Discord Pokémon kart ve savaş botudur. Arkadaşlarınızla savaşabilir, vahşi rakiplere karşı mücadele edebilir, kart toplayabilir ve eğlenceli vakit geçirebilirsiniz.

## Önemli Notlar:
### Gelecekte bu bota güvenlik botu özelliği ve sansür botu olarak güncellemeler planlanıyor
### Ekibimiz sizin en iyi deneyiminiz olmasını sağlamak için düzenli olarak güncelliyoruz. eğer bir hata görürseniz lütfen bildirin hatayı düzeltelim

# 🎮 POKÉMON KART BOTU KULLANIM KILAVUZU

## 🚀 Başlangıç
`!go` → Pokémon oluştur  
`!info` → Bilgini gör  
`!info @kişi` → Başkasının Pokémonunu gör  

## 🍗 Gelişim
`!feed` → Besle (XP kazanırsın, level atlarsın)  
`!heal` → Can yenile  
`!daily` → Günlük coin al  

## 🛒 Market & Kartlar
`!shop` → Marketi gör  
`!buy kart_id miktar` → Kart satın al  

Örnek:  
```bash
!buy heal 2
```

`!cards` → Kartlarını gör  

## 🎴 Kartlar
`power` → Ekstra hasar  
`heal` → Can yeniler  
`shield` → Hasarı azaltır  
`rage` → Sonraki saldırıyı güçlendirir  

## ⚔️ Savaş
`!battle` → Vahşi rakiple savaş  
`!battle @kişi` → Arkadaşla savaş  

### Savaş Butonları
⚔️ Saldır → Normal vur  
🛡️ Savun → Hasarı azalt  
🎴 Kartlar → Güç kullan  
🏃 Kaç → (Sadece vahşi savaşta)  

## 💰 Takas & Para
`!sendmoney @kişi miktar` → Para gönder  

Örnek:  
```bash
!sendmoney @Emin 50
```

`!tradecard @kişi kart_id miktar` → Kart gönder  

Örnek:  
```bash
!tradecard @Emin heal 1
```

## 📌 Notlar

• Herkes sadece 1 Pokémon alabilir  
• Kartlar savaşta avantaj sağlar  
• Level arttıkça güçlenirsin  
• Bot kapanınca veriler silinmez
